
# GitHub Secrets Setup for Firebase & Production Deployments

Replit-рЃАрЃЌрЃљрЃю рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃљрЃдрЃљрЃа рЃњрЃЋрЃљрЃЦрЃЋрЃА. Production рЃЊрЃљ CI secrets рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ GitHub Repository Secrets-рЃерЃў рЃљрЃЏ рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ:

## 1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ GitHub Secrets рЃњрЃЋрЃћрЃарЃЊрЃў
- рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃќрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџ рЃЉрЃЏрЃБрЃџрЃќрЃћ (Actions secrets)
- рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **"New repository secret"** рЃдрЃўрЃџрЃљрЃЎрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

## 2. рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ Firebase-рЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў

### 2.1 рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў frontend рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў
`.env.example`-рЃерЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЏрЃљрЃарЃЌрЃљрЃЋрЃћрЃю рЃљрЃЊрЃЏрЃўрЃюрЃўрЃА рЃЉрЃБрЃЌрЃАрЃбрЃарЃљрЃърЃА, WebAuthn-рЃАрЃљ рЃЊрЃљ AI рЃЕрЃљрЃбрЃўрЃА рЃарЃћрЃЪрЃўрЃЏрЃћрЃЉрЃА. рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЌ рЃўрЃАрЃўрЃюрЃў GitHub Secrets-рЃерЃў:

- `VITE_ADMIN_SETUP_TOKEN` Рђћ рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃљрЃЊрЃЏрЃўрЃюрЃўрЃА рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ (рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃў: `ai-frontend/src/utils/adminToken.ts`).
- `VITE_ORIGIN` / `VITE_RP_ID` Рђћ WebAuthn passkey рЃЊрЃЮрЃЏрЃћрЃюрЃћрЃЉрЃў (`ai-frontend/src/utils/webauthn_support.ts`).
- `VITE_ENABLE_PUBLIC_CHAT` Рђћ рЃАрЃбрЃБрЃЏрЃарЃўрЃА рЃЕрЃљрЃбрЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃљ (`ai-frontend/src/components/AIAssistantEnhanced.tsx`).
- `VITE_ASSISTANT_MODE` Рђћ рЃљрЃАрЃўрЃАрЃбрЃћрЃюрЃбрЃўрЃА рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃарЃћрЃЪрЃўрЃЏрЃў (`ai-frontend/src/contexts/AssistantModeContext.tsx`).
- `VITE_GITHUB_ENABLED` Рђћ GitHub workspace-рЃўрЃА рЃбрЃЮрЃњрЃџрЃў (`ai-frontend/src/lib/featureFlags.ts`).
- `VITE_BACKEND_URL` / `VITE_API_BASE` / `VITE_GATEWAY_URL` Рђћ backend/gateway рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃў (`ai-frontend/src/lib/env.ts`).
- `AI_SERVICE_URL` Рђћ API proxy fallback рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў (`ai-frontend/pages/api/ai/[...path].ts`).

## 3. рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЌ GitHub Actions рЃњрЃљрЃарЃћрЃЏрЃЮ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў
- рЃЌрЃБ рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃўрЃљ workflow-рЃћрЃЉрЃў, рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃЏрЃљрЃЌрЃў secrets рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃарЃЮрЃЏ рЃљрЃ«рЃљрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЕрЃљрЃўрЃбрЃЋрЃўрЃарЃЌрЃЮрЃА
- рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `node check-secrets.js` рЃљрЃЊрЃњрЃўрЃџрЃЮрЃЉрЃарЃўрЃЋрЃљрЃЊ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃџрЃљрЃЊ
- рЃљрЃ«рЃљрЃџрЃў Operations рЃърЃљрЃюрЃћрЃџрЃў рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃА GitHub smoke рЃбрЃћрЃАрЃбрЃўрЃА рЃАрЃћрЃЎрЃарЃћрЃбрЃћрЃЉрЃА (рЃЌрЃБ рЃњрЃАрЃБрЃарЃЌ sandbox PR-рЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃЦрЃЏрЃюрЃљ):
  - `GITHUB_SANDBOX_OWNER`
  - `GITHUB_SANDBOX_REPO`
  - `GITHUB_SANDBOX_BASE_BRANCH` (рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа)

## 4. рЃЋрЃћрЃарЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ
- Frontend build-рЃўрЃАрЃљрЃА рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃерЃў рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃА: "РюЁ Firebase initialized successfully"
- `node scripts/github-verification.js` рЃЊрЃљрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃА GitHub Token-рЃАрЃљ рЃЊрЃљ webhook secret-рЃА

## Troubleshooting
- рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃфрЃЋрЃџрЃљрЃЊрЃў рЃўрЃгрЃДрЃћрЃЉрЃљ `VITE_` рЃърЃарЃћрЃцрЃўрЃЦрЃАрЃўрЃЌ (frontend-рЃАрЃЌрЃЋрЃўрЃА)
- `FIREBASE_SERVICE_ACCOUNT_KEY` рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃарЃБрЃџрЃў JSON (рЃћрЃарЃЌ рЃ«рЃљрЃќрЃќрЃћ рЃљрЃю рЃЏрЃарЃљрЃЋрЃљрЃџ рЃ«рЃљрЃќрЃљрЃЊ)
- GitHub Secrets рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ whitespace рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃГрЃарЃћрЃЉрЃљ; рЃфрЃљрЃарЃўрЃћрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃА рЃерЃћрЃбрЃљрЃюрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃћрЃЋрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃА
